How to Horizontally Center an Element in CSS: 5 Modern Methods + Browser Support Guide

Quick Answer

To horizontally center a div, use justify-content: center for modern browsers, or margin: 0 auto with a defined width for universal compatibility.

/* Modern approach */
.center { display: flex; justify-content: center; }

/* Classic approach (universal) */
.center { width: 50%; margin: 0 auto; }

Choose Your Method

Need to know the element's width?

  • YES → Use Flexbox | Method 1 | → NO → Use margin auto | Method 2
  • Need both horizontal & vertical? → YES → Use CSS Grid | Method 5

Method 1: Flexbox (Recommended)

.container { display: flex; justify-content: center; }
/* Universal: Margin Auto */
.centered-div { width: 300px; /* or any fixed width */ margin: 0 auto; }

Method 2: Margin Auto (Classic)

.centered-div { 
  width: 300px; /* or any fixed width */ 
  margin: 0 auto; 
}

Centering a Card Component

.card {
  width: 400px;

Summary

  • Use Flexbox for modern, responsive centering
  • Use margin auto for simple, universal compatibility
  • Use CSS Grid for complex two-dimensional layouts
  • Always test your chosen method across target browsers
