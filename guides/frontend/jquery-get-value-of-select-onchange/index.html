jQuery Get Value of Select onChange: 5 Methods + Performance Guide

Quick Answer

To get the value of a select element when it changes in jQuery, use $(this).val() inside a change event handler:

$('select').on('change', function() {


    console.log(this.value);


});

For modern JavaScript, you can also use event.target.value with arrow functions.

Method 1: jQuery .on() with this.value (Recommended)

This is the most efficient and widely-used method for getting select values on change.

$('select').on('change', function() {


    console.log(this.value);


});

Complete Example:

Advantages:

  • Fastest performance (this.value is native JavaScript)
  • Works with jQuery 1.7+
  • Clean, readable syntax
  • Supports event delegation

When to use:

  • Modern jQuery applications
  • Performance-critical scenarios
  • Multiple select elements

Method 2: jQuery .on() with $(this).val()

This method uses jQuery's .val() method instead of native JavaScript.

$('select').on('change', function() {


    alert($(this).val());


});

Complete Example:

Advantages:

  • Consistent with jQuery patterns
  • Handles edge cases automatically
  • More readable for jQuery developers

When to use:

  • jQuery-heavy applications
  • When consistency with jQuery patterns is important
  • Complex form handling

Method 3: jQuery .change() Method

The traditional jQuery method for handling change events.

$('#select-id').change(function() {


    console.log($(this).val());


});

Complete Example:

Advantages:

  • Simple syntax
  • Works with older jQuery versions
  • Familiar to most developers

When to use:

  • Legacy jQuery applications
  • Simple event handling
  • Older browser support needed

Method 4: Arrow Functions with event.target

Modern ES6+ approach using arrow functions.

$('select').on('change', (event) => {


    console.log(event.target.value);


});

Complete Example:

Important Note: Arrow functions don't bind this, so you must use event.target instead of this.

Advantages:

  • Modern JavaScript syntax
  • Consistent with ES6+ patterns
  • Explicit event parameter

When to use:

  • Modern applications using ES6+
  • When mixing with other arrow functions
  • React/Vue.js integration

Method 5: Event Delegation

Perfect for dynamically added select elements.

$(document).on('change', 'select', function() {


    console.log(this.value);


});

Complete Example:

Advantages:

  • Works with dynamically added elements
  • Single event handler for multiple selects
  • Memory efficient

When to use:

  • Dynamic content (AJAX-loaded forms)
  • Single-page applications
  • Multiple similar select elements

Common Problems & Solutions

Problem 1: this is undefined in arrow functions

Issue:

$('select').on('change', () => {


    console.log(this.value); // undefined!


});

Solution:

$('select').on('change', (event) => {


    console.log(event.target.value); // ✅ Works


});

Problem 2: Event not firing on dynamically added selects

Issue:

// This won't work for future elements


$('.dynamic-select').on('change', function() {


    console.log(this.value);


});

Solution:

// Use event delegation


$(document).on('change', '.dynamic-select', function() {


    console.log(this.value);


});

Problem 3: Getting text instead of value

Issue:

// This gets the display text, not the value


var text = $(this).text();

Solution:

// Get value


var value = $(this).val();


// Get selected option text


var text = $(this).find(':selected').text();

Problem 4: Multiple select handling

Issue:

// Only gets first value in multi-select


var value = $(this).val();

Solution:

// Get all selected values


var values = $(this).val(); // Returns array for multi-select


console.log(values); // ['value1', 'value2']

Problem 5: Firefox keyboard navigation

Issue: Firefox doesn't trigger change event when using arrow keys.

Solution:

$('select').on('change keyup', function() {


    console.log(this.value);


});

Example 1: Dependent Dropdowns

// Country/State selection


$('#country').on('change', function() {


    const countryId = this.value;


    const stateSelect = $('#state');



    // Clear and disable state dropdown


    stateSelect.empty().prop('disabled', true);



    if (countryId) {


        // Load states via AJAX


        $.ajax({


            url: /api/states/${countryId},


            success: function(states) {


                stateSelect.prop('disabled', false);


                states.forEach(state => {


                    stateSelect.append();


                });


            }


        });


    }


});

Example 2: Form Validation

$('.required-select').on('change', function() {


    const value = this.value;


    const $this = $(this);



    if (value) {


        $this.removeClass('error').addClass('valid');


        $this.next('.error-message').hide();


    } else {


        $this.removeClass('valid').addClass('error');


        $this.next('.error-message').show();


    }



    // Check if form is complete


    validateForm();


});



function validateForm() {


    const allValid = $('.required-select').toArray().every(select => select.value);


    $('#submitBtn').prop('disabled', !allValid);


}

Example 3: Price Calculator

$('.product-option').on('change', function() {


    calculateTotal();


});



function calculateTotal() {


    let total = 0;



    $('.product-option').each(function() {


        const value = this.value;


        const price = $(this).find(':selected').data('price') || 0;


        total += parseFloat(price);


    });



    $('#total-price').text($${total.toFixed(2)});


}

Example 4: Filter System

$('.filter-select').on('change', function() {


    const filters = {};



    // Collect all filter values


    $('.filter-select').each(function() {


        const filterName = $(this).data('filter');


        const filterValue = this.value;



        if (filterValue) {


            filters[filterName] = filterValue;


        }


    });



    // Apply filters


    applyFilters(filters);


});



function applyFilters(filters) {


    $('.product-item').each(function() {


        const $item = $(this);


        let show = true;



        Object.keys(filters).forEach(filterName => {


            const itemValue = $item.data(filterName);


            if (itemValue !== filters[filterName]) {


                show = false;


            }


        });



        $item.toggle(show);


    });


}

Summary/Key Takeaways

  2. Use $(this).val() for consistency with jQuery patterns
  3. Use event delegation for dynamically added select elements

  7. Use proper selectors (ID vs class vs element) based on your needs

Frequently Asked Questions

Q: Why does my query return undefined?

A: This usually happens with arrow functions. Arrow functions don't bind this:

// ❌ Wrong - this is undefined


$('select').on('change', () => {


    console.log(this.value); // undefined


});



// ✅ Correct


$('select').on('change', (event) => {


    console.log(event.target.value);


});

Q: Which method is fastest for large tables?

A: Use this.value with event delegation:

$(document).on('change', 'select', function() {


    console.log(this.value); // Fastest


});

Q: How do I handle NULL or empty values?

A: Always check for empty values:

$('select').on('change', function() {


    const value = this.value;


    if (value && value !== '') {


        // Process non-empty value


        console.log('Selected:', value);


    } else {


        // Handle empty selection


        console.log('No selection');


    }


});

Q: Why isn't the change event firing?

A: Common causes:

  1. Dynamic elements: Use event delegation
  2. Programmatic changes: Trigger change manually
  3. Firefox keyboard navigation: Add keyup event

// For dynamic elements


$(document).on('change', 'select', function() { / ... / });



// For programmatic changes


$('#mySelect').val('newValue').trigger('change');



// For Firefox keyboard support


$('select').on('change keyup', function() { / ... / });

Q: How do I get both value and text?

A: Use both .val() and :selected selector:

$('select').on('change', function() {


    const value = $(this).val();


    const text = $(this).find(':selected').text();


    console.log(Value: ${value}, Text: ${text});


});

Q: Can I use this with React or Vue.js?

A: While possible, it's better to use framework-specific methods:

// React





// If you must use jQuery in React/Vue


useEffect(() => {


    $('#mySelect').on('change', function() {


        console.log(this.value);


    });



    return () => $('#mySelect').off('change');


}, []);
